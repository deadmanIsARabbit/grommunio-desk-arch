# grommunio-desk-arch
PKGBUILD for [grommunio-desk](https://github.com/grommunio/grommunio-desk) on ARCH Linux

## Requirements

- Node.js 22.* to 24.* (above will fail)

## Installation

Either: \
`cd /tmp/` \
\
`wget https://raw.githubusercontent.com/deadmanIsARabbit/grommunio-desk-arch/refs/heads/main/PKGBUILD` \
\
`makepkg -si`

OR:

`git clone https://github.com/deadmanIsARabbit/grommunio-desk-arch.git` \
\
`cd grommunio-desk-arch` \
\
`makepkg -si`
